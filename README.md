# Pruebas en ESP_IDF

Pequeño repositorio con ejemplos y prácticas programando en ESP_IDF.

##  Objetivos

- Ampliar conocimientos en programación en sistemas embebidos.
- Uso de diversas herramientas.
- Escala a futuro de otros microcontroladores.

##  Proceso:

Iniciación de proyectos básicos buscando una mejor optimización trabajando en este entorno FreeRTOS reemplazando los IDE como Arduino IDE, y explorar las capacidades disponibles.
